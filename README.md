# Учёт финансов фирмы по продаже автозапчастей

## Предметная область
Фирма занимается продажей запасных частей для автомобилей.  
Основная задача – отслеживание финансовой деятельности, связанной с закупками у поставщиков.

## Сущности
- **Поставщик** – название, адрес, телефон.
- **Деталь** – название, артикул, цена (постоянная).
- **Покупка** – дата покупки, количество деталей, поставщик, деталь.

## Функциональность
- Фиксация каждого факта покупки запчастей.
- Возможность анализа: какие детали и у каких поставщиков закупаются.
- Учёт постоянной цены детали для финансовых отчётов.

## Структура репозитория
- `Общее_Описание.txt` – детальное описание базы данных и бизнес-процессов.
- `.gitignore` – исключение временных и служебных файлов.
- Ветка `extra` – дополнительные материалы по методам и рекомендациям распределения закупок.
